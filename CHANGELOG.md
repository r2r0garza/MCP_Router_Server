# Changelog

## v1.0.0

- Initial release of MCP Router Server
- Supports OpenAI, LM Studio, OpenRouter, Ollama, Anthropic Claude, Azure Foundry AI providers
- Unified `/ask` endpoint with context schema validation
- Health check endpoint
- In-memory context store
- Dockerfile and CI workflow
- Python client (`mcp_client`)
- Example `.http` and `curl` scripts
